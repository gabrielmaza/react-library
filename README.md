## Install

You must have Node.js installed and import the next github backend project:
https://github.com/gabrielmaza/library-node-backend.git

To start
```
node install
```

For running
```
npm start
```

# Api endpoints.

## Api categoria
```
GET     /api/categoria 

POST    /api/categoria 

GET     /api/categoria/:id 

DELETE  /api/categoria/:id 
```
## Api Persona
```
GET     /api/persona 

POST    /api/persona 

GET     /api/persona/:id 

PUT     /api/persona/:id 

DELETE  /api/persona/:id 
```
